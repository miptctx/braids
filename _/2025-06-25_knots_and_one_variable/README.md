В этих рассчетах проверялась конструкция с движениями 1-3, 3-1 и 2-2 для узла внутри треугольника.

В результате оказалось, что в такой конструкции умножение всех матриц соответствующих какому-либо узлу равно [1].

Фактически получился тривиальный инвариант, принимающий значение 1 для всех узлов.

Это все же лучше, чем разные значения для разной изотопии одного и того же узла.

Можно подумать, как можно улучшить эту конструкцию.

Также проверял соотношение Янга-Бакстера если переименовывать точки после каждой дзеты. Но оно не выполняется.

Соотношение для кос, если точки не переименовывать, выполняется нормально.
