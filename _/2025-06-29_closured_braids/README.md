Пробовал пересчитать систему линейных уравнений, которая составляется для замкнутых кос.

Составлял две системы. Первая имел вид x=xA, вторая - E=A.

В зависимости от того, каким именно способом решать эту систему, получаются либо тривиальное решение из одних единиц для разных узлов, либо решения получаются неединственными.

Для восьмерки система E=A относительно дзет не имеет решений, а для трилистника на 2 нитях решенеие выдает одни единицы. Поэтому система E=A возможно не самая правильная.

Для системы вида x=xA решение тривиальное для всех узлов - все компоненты вектора (x_1,x_2,...,x_n) равны единицам (либо просто равны между собой и зависят от произвольного числа или чисел).

решения полученные с помощью функции solve_right наверное не имею никакого смысла, т.к. они решают систему уравнений вида Ax=b, где b - это верктор содержащий единицы в наших рассчетах, но нам нужно, чтобы он содержал переменную. Но решения я оставил, т.к. они получаются единичными, что тоже интересно.

Составление системы уравнений x=xA означает, что мы отождествляем векторные пространства в начале косы и в конце косы. Мы говорим, замыкание косы означает, что пространсвоа в начале и в конце - это одно и то же пространство с одним и тем же базисом, а матрица становится матрицей линейного оператора. И кроме этого мы приравниваем образы векторов этого линейного оператора к базисным векторам векторного пространства, после чего получаем систему уравнений.

Решения x=xA
------------
Тривиальный узел:
$$
t_125 == r1, t_134 == r1, t_145 == r1, t_234 == r1, t_245 == r1
$$

Трилистник с двумя нитками:
$$
t_125 == r1, t_134 == r1, t_145 == r1, t_234 == r1, t_245 == r1
$$

Пятерка с двумя нитями
$$
t_125 == r1, t_134 == r1, t_145 == r1, t_234 == r1, t_245 == r1
$$

Тривиальный узел с 1 движением Маркова
$$
t_126 == r2, t_134 == r2, t_145 == r2, t_156 == r2, t_234 == r2, t_245 == r2, t_256 == r1
$$

Трилистник на двух нитях с 1 движением Маркова
$$
t_126 == 2*r1 - r2, t_134 == 2*r1 - r2, t_145 == 2*r1 - r2, t_156 == r2, t_234 == 2*r1 - r2, t_245 == 2*r1 - r2, t_256 == r1
$$

Трилистник на трех нитях
$$
t_126 == r1, t_134 == r1, t_145 == r1, t_156 == r1, t_234 == r1, t_245 == r1, t_256 == r1
$$

Восьмерка на трех нитях
$$
t_126 == r2, t_134 == r2, t_145 == r2, t_156 == 6/5*r1 - 1/5*r2, t_234 == r2, t_245 == r2, t_256 == r1
$$

Восьмерка на 3 нитях и 1 движением
$$
t_127 == r3, t_134 == r3, t_145 == r3, t_156 == -2*r1 + 6/5*r2 - 6/5*r3 + 3*r4, t_167 == r4, t_234 == r3, t_245 == r3, t_256 == r2, t_267 == r1
$$

Исходя из полученных систем ничего особо хорошего сказать нельзя. Во-первых, решения для тривиального узла на двух нятях такое же как и для трилистника и пятерки на двух нитях. Это означает, что такой подход уже не различает эти узлы, что делает его использование бесмыссленным.
Еще решения системы для трилистника на трех нитях выглядит так же, как и для трилистника на двух нитях. Но при этом решение для системы трилистника с одним движением Маркова выглядит по другому и это тоже плохо.
